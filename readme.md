E-Commerce Revenue Tracker Dashboard

The E-Commerce Revenue Tracker Dashboard is an interactive data analytics solution designed to visualize and monitor key performance metrics of an e-commerce business. Built using IBM Cognos Analytics, with interface support from HTML and CSS, the dashboard transforms raw sales data into meaningful insights to support real-time, data-driven decision-making.
