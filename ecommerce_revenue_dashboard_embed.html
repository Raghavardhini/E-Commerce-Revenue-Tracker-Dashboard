E-Commerce Revenue Tracker
This project demonstrates a revenue analytics dashboard embedded from IBM Cognos Analytics. Use the "Open Dashboard" button to view the live report. The embedded view hides Cognos' appbar and navbar for a clean presentation.

About this dashboard

Purpose: Visualize revenue trends and KPIs for decision makers. Data sources, filters, and drill-throughs remain available inside the embedded Cognos view. The iframe below points at the "Revenue" subview and uses embedded mode parameters so it can be cleanly placed inside other pages.

Tip: If your organization requires single sign-on, ensure the user session is authenticated in the browser before loading the iframe.
